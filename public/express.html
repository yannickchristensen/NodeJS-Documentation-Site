Express

What is Express?

Express is a back end web application framework for NodeJS. Express allows us to create URLs/endpoints that the user can navigate to in the browser. We can then define code to handle those requests.

How to install express?

Express is also very easy to install as we will just use npm for this as well. All you have to do is type "npm install express". Two things will happen; a directory containing the source code to your dependencies will be added to your main directory called node_modules and express with corresponding version number has been added to your package.json file, see below.

How to build your first webapp with express?

It's very simple to wire a web app with Express and NodeJS. This is the setup for the home page in this web application, you'll see it's only a few lines of code.


            const express = require('express');                         // imports the express module

            const app = express();                                      // initialize express app
            
            app.get('/', (req, res) => {                                // create endpoint for home, when a user calls our browser with this endpoint we will serve them an html file called home.html
                res.sendFile(__dirname + "/public/home.html");
            });
