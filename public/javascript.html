JavaScript

What is JavaScript?

JavaScript is a programming language that was first invented as a scripting language for implementing complex features on the web. One way to this is DOM (Document Object Model) methods, with the object model JavaScript gets the power to create dynamic HTML. Javascript in conjuction with the DOM can change HTML elements, attributes and CSS styles.

Data types and dynamic typing:

JavaScript is a loosely typed and dynamic language. Variables in JavaScript are not directly associated with any particular value type, and any variable can be assigned (and re-assigned) values of all types:


let foo = 42;    // foo is now a number
foo     = 'bar'; // foo is now a string
foo     = true;  // foo is now a boolean

Declaring variables:

There are 3 ways to declare a JavaScript variable:

  • Using var
  • Using let
  • Using const

    • What's the difference between the three ways?

  • var declarations are globally scoped or function scoped while let and const are block scoped.(Block scoping means declaring a variable, not just inside a function, but around any curly brackets like if statements or loops.)
  • var variables can be updated and re-declared within its scope. let variables can be updated but not re-declared. const variables can neither be updated nor re-declared.
  • They are all hoisted to the top of their scope. But while var variables are initialized with undefined, let and const variables are not initialized.
  • While var and let can be declared without being initialized, const must be initialized during declaration.

    • Functions in JavaScript:

    In javascript functions are first classed citizens which means that functions supports all the operational properties inherent to other entities. Therefore we can:

    Assign a function to a variable:
    
var func = function (param) {
    return param;
}
variable += func(param);
    Pass a function as an argument to another function and return a function from a function:
    
var func = function (param) {
    return param;
}
                      
function func1 (param1) {
return 10;
}

func(func1); // ==10
    Callback functions

    In javascript a callback is a function that is passed as an argument to another function. In the code snippet below, the arrow function is a callback function. An arrow function provides a shorter syntax over regular functions:

    
app.get('/', (req, res) => {
    res.sendFile(__dirname + "/public/home.html");
});
