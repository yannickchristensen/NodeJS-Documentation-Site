NodeJS

What is NodeJS?

NodeJS is a runtime environment for JavaScript which makes it possible to build full stack web applications entirely in JavaScript. Before NodeJS was introduced in 2009 it was impossible to write serverside JavaScript code. Now we can use NodeJS to handle HTTP requests, read HTML file in the servers filesystem and then send a respond back to the client so they can view the HTML file in their browser.

How to install NodeJS?

  1. Download the installer from the NodeJS website.
  2. Run the installer.
  3. Follow the installer steps, agree the license agreement and click the next button.
  4. Restart your system/machine.

What is nodemon?

Nodemon is a utility that will monitor for any changes in your source directory and automatically restart your server. This is convenient when developing as you can see the changes you made straight away.

How to install nodemon?

Install nodemon by using npm with following command : "npm install -g nodemon".

You can also install nodemon as a deployment dependency. Inside your project directory type:"npm install --save-dev nodemon" in your terminal. This will add nodemon to your devDependencies in your package.json file. This will allow you to create a script to run nodemon which you can configure as you like. 


        {
          "name": "NodeJS-Documentation-Site",                                  
          "version": "1.0.0",
          "description": "",
          "main": "index.js",
          "scripts": {
            "watch": "nodemon",                                           //nodemon script
            "test": "echo \"Error: no test specified\" && exit 1"
          },
            "keywords": [],
            "author": "",
            "license": "ISC",
            "devDependencies": {
              "nodemon": "^2.0.13"                                        //nodemon added is dependency
            }
        }

You can run this watch script using npm with this command: "npm run watch", because we have set index.js as our main we don't have to specify which script to run.
